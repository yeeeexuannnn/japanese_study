# Quickstart & Verification Guide: 日語學習工具 (Japanese Learning Tool)

本指南介紹如何在您的 Windows 開發機上啟動本地伺服器，並在行動端（Android 手機、iOS 平板）連接、進行安裝 (PWA) 與功能測試。

## 1. 環境準備 (Prerequisites)

*   本機需安裝有 [Python 3](https://www.python.org/)。
*   行動裝置（手機或平板）必須與您的開發機（電腦）連線在 **同一個 Wi-Fi 無線網路環境** 下。

---

## 2. 啟動開發伺服器 (Start Local Server)

開啟電腦的終端機（PowerShell 或 CMD），進入專案根目錄，執行以下指令啟動一個簡單的靜態網頁伺服器：

```bash
# 使用 Python 內建的 HTTP 伺服器啟動於 8000 埠
python -m http.server 8000
```

---

## 3. 在行動端（手機/平板）訪問與安裝 PWA

### 步驟 A：取得本機 IP 位址
在電腦上打開 PowerShell，輸入：
```powershell
ipconfig
```
尋找您的 `IPv4 位址`（例如 `192.168.1.104`）。

### 步驟 B：在手機/平板上打開瀏覽器網頁
1.  **iOS 平板 (Safari)：** 打開 Safari，在網址列輸入 `http://192.168.1.104:8000`。
2.  **Android 手機 (Chrome)：** 打開 Chrome，輸入 `http://192.168.1.104:8000`。

### 步驟 C：安裝到主畫面 (PWA)
*   **iOS/iPadOS (Safari)：** 點選瀏覽器底部的「分享」按鈕 ➡️ 選擇 **「加入主畫面」(Add to Home Screen)**。
*   **Android (Chrome)：** 點選右上角選單（三個點）➡️ 選擇 **「安裝應用程式」** 或 **「加到主畫面」**。

此時您的平板/手機桌面會出現一個名為「大家的日本語2 單字練習」的圖示，點選開啟將以全螢幕（無瀏覽器網址列）呈現。

---

## 4. 端到端功能驗證情境 (End-to-End Verification Scenarios)

### 情境一：3D 單字卡學習功能驗證
1.  進入首頁，在「範圍選擇」中勾選「第 13 課」。
2.  點擊進入「單字卡學習模式」。
3.  預期結果：畫面顯示第一個單字「遊びます」。點擊卡片後，卡片進行 3D 翻轉，顯示讀音 `あそびます`、翻譯 `玩、遊玩`。點選右下角的「⭐ 星星」按鈕。
4.  關閉網頁並重新整理，再次進入，確認「遊びます」的星星圖示仍保持亮起狀態（已收藏）。

### 情境二：看日選中（選擇題）測驗
1.  在首頁範圍勾選「第 13 課」，選擇「看單字選中文翻譯」模式，點擊「開始測驗」。
2.  系統隨機顯示一個單字，並附帶四個不重複的中文選項。
3.  點擊正確答案，預期結果：題目卡片瞬間以綠色邊框/背景提示「正確」，答對題數 +1。

### 情境三：書籤範圍測驗
1.  承接情境一，確保您已收藏「遊びます」。
2.  回到首頁，在範圍選擇中取消所有課別，**僅勾選「書籤（標星單字）」**。
3.  選擇「中翻日」或「日翻中」模式並開始測驗。
4.  預期結果：測驗中的題目僅會出現「遊びます」這一道題。
